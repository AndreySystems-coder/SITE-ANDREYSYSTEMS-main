# SITE-ANDREYSYSTEMS-main

Landing page profissional para a marca `ANDREY.SYSTEMS`, criada com estética dark premium, editorial e minimalista para serviços de websites, sistemas, ERPs, automações e consultoria.

## Estrutura

- `index.html`: estrutura principal da landing page.
- `styles.css`: identidade visual inspirada na referência enviada, responsividade e estados de hover.
- `script.js`: menu mobile, ano automático no rodapé e animações leves de entrada.
- `assets/images/premium-workspace.png`: visual editorial principal do hero e case ERP IMPPEL.
- `assets/images/erp-imppel-showcase.png`: asset anterior mantido no projeto para uso futuro, mas não usado no layout atual.

## Como rodar

Opção simples:

1. Abra `index.html` diretamente no navegador.

Opção com servidor local sem instalar dependências:

```bash
node server.mjs
```

Depois acesse:

```text
http://127.0.0.1:4217
```

Opção com Vite:

```bash
npx vite . --host 127.0.0.1 --port 5173
```

Depois acesse:

```text
http://127.0.0.1:5173
```
